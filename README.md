Arithmetic Calculator

A simple Arithmetic Calculator built using HTML, CSS, and JavaScript. This calculator allows users to perform basic arithmetic operations like addition, subtraction, multiplication, and division in a user-friendly interface.

Features

Perform addition, subtraction, multiplication, and division

Clear input with the Clear (C) button

Responsive and clean user interface

Built using vanilla JavaScript, HTML, and CSS

Technologies Used

HTML – For structure of the calculator

CSS – For styling and layout

JavaScript – For calculator logic and interaction

How to Use

Open the index.html file in your browser.

Enter numbers and select arithmetic operations.

Click = to get the result.

Click C to clear the input.
